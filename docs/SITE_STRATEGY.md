# サイト戦略 — SITE_STRATEGY

## 1. サイトの役割

城北セレモニーサポートセンターは、**北区・板橋区の方が戸田斎場をはじめとする斎場を利用してご葬儀を行う際の相談窓口**サイトである。運営・施行は川口典礼。
電話相談（0120-963-765）を主CTAとし、「まずは戸田斎場で考えているとお電話ください」という入口を提供する。

## 2. 川口典礼本体サイトとの役割分担

| | 本体サイト（川口典礼） | 本サイト（城北セレモニーサポートセンター） |
|---|---|---|
| 主エリア | 川口市・川口エリア | 東京都北区・板橋区 |
| 主な受け皿KW | 川口 葬儀／川口市 家族葬／川口典礼／めぐりの森／川口メモリアルホール | 北区・板橋区 ＋ 葬儀／家族葬／一日葬／火葬式／直葬、各斎場名 |
| 役割 | ブランド本体・川口市の集客 | 北区・板橋区および戸田斎場ほか斎場名の検索意図を受ける |

**カニバリ回避**：本サイトでは川口市・めぐりの森・川口メモリアルホールを主キーワードにしない。
本体への導線はフッター／運営者情報からの控えめなリンクに留める。

## 3. 北区・板橋区向けにする理由

- 北区・板橋区は、火葬場併設の戸田斎場をはじめ近隣斎場の利用ニーズが高い。
- 「北区 葬儀」「板橋区 家族葬」等の地域 × 形式の検索意図に、本体サイト（川口市軸）では十分に応えられない。
- 地域特化の相談窓口として分離することで、各エリア・各斎場の検索意図を明確に受け止められる。

## 4. 戸田斎場を中心にする理由

- 戸田斎場は火葬場併設で、式場〜火葬炉の移動が少なく、北区・板橋区からアクセスしやすい。
- 一日葬・火葬式・直葬との相性がよく、相談ボリュームが大きい。
- そのため戸田斎場ページを最重要ページと位置づけ、トップ・エリア・プランから内部リンクを集約する。

## 5. 斎場別ページ戦略

対象斎場：戸田斎場／蓮根レインボーホール／北区セレモニーホール／舟渡斎場。

- 各斎場ページは「斎場名 × 葬儀／一日葬／火葬式／家族葬／直葬」の検索意図を受ける。
- 構成は共通（H1・リード・相談が多い形式・利用の流れ・搬送/安置/空き確認・費用の考え方・FAQ・CTA・関連リンク）だが、**本文は斎場ごとに固有の文章**にし、コピー量産を避ける。
- 各斎場ページに「公式施設サイトではなく運営・施行は川口典礼」の注記を必ず入れる。
- データは `data/halls.ts` に集約し、ページは動的ルート `app/hall/[hall]/page.tsx` で生成する。

## 6. ページ階層と受け皿

- トップ：ブランド ＋ 北区/板橋区 葬儀 ＋ 戸田斎場 一日葬/火葬式 の総合入口。
- エリア（北区／板橋区）：地域 × 形式。地域内の斎場と各プランへ送客。
- 斎場（4施設）：斎場名 × 形式。プランとエリアへ相互リンク。
- プラン（一日葬／火葬式・直葬／家族葬）：形式 × 地域・斎場の横断受け皿。
- FAQ：AIO 受け皿（FAQPage）。
