# コピーライティングガイド — COPYWRITING_GUIDE

葬儀業界向けに、顧客心理へ寄り添い、不安を解消し、電話につながる文章を書く。

## 1. 葬儀業界向けの表現ルール

- 誠実・落ち着いた敬体。煽らない。断定しすぎない。
- 「お見送り」「お別れ」「ご相談」など、配慮のある言葉を使う。
- 価格は主役にしすぎず、「何が含まれ」「何で変わり」「相談で整理できる」を必ず添える。

## 2. 顧客心理（前提）

- 何から始めればよいか分からない。
- 戸田斎場を使いたいが流れが分からない。
- 斎場の空きが不安。
- 病院・施設からどこへ搬送すればよいか分からない。
- 一日葬・火葬式・家族葬の違いが分からない。
- 総額がいくらになるか不安。
- 強い営業をされたくない。
- 公共施設・民間斎場の違いが分からない。

## 3. 不安解消型コピー（使ってよい表現）

- まずは「戸田斎場で考えている」とお電話ください
- 斎場の空き確認から、搬送・安置・お見積りまでご相談いただけます
- 北区・板橋区から利用しやすい式場を、状況に合わせてご案内します
- 通夜を行わず、一日でお見送りする形式です
- 火葬を中心に、できるだけシンプルにお見送りする形式です
- ご家族中心で、落ち着いてお別れする形式です
- 費用は内容や人数、式場、返礼品、飲食の有無などにより変わります

## 4. 電話につながるコピー

- CTAは「今すぐ申込」ではなく「まずは相談」「空き確認」「見積り相談」。
- 電話の心理的ハードルを下げる枕詞：「まずは」「〜だけでも」「お気軽に」。
- 24時間365日受付・ご相談お見積り無料を、煽らずに明示する。
- 「何と言って電話すればいいか」を提示する（例：「戸田斎場で考えている」とお伝えください）。

## 5. 価格表現ルール

- 価格は「目安」「税込」で表現する。
- 「内容により変動する」「正確な費用は個別にお見積り」を必ず併記する。
- 式場料金・火葬料金・宗教者へのお礼・返礼品・飲食費などは別途／内容により変動する旨を添える。
- 実装は `components/PriceNote.tsx`（注記の共通コンポーネント）と `app/config/site.ts` の `priceNotes`。

掲載している目安価格（税込）：
- 一日葬：496,000円
- 火葬式・直葬：189,000円
- 家族葬：628,000円

## 6. 禁止表現

最安／激安／必ず／絶対／追加費用なし／No.1／どこよりも安い／公式／公認／
北区民のみ／板橋区民のみ／競合批判／不安を過度に煽る表現。

## 7. 必ず明記する一文

> 城北セレモニーサポートセンターは各斎場の公式施設サイトではありません。運営・施行は川口典礼です。

FAQ・斎場ページ・運営者情報など、必要箇所に明記する。
